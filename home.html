Introduction

Welcome to flowork, a simple todo web app for those requiring simplicity and a clean interface.

The simple and clean interface helps you stay clean, frustration free.

Flowork's user friendly interface helps you keep yourself organized.

Features

Flowork provides a neat UX along with:

  • Simple and clean interface.

  • Single page UX, thus hassle free.

  • Ability to strike todos.

  • Responsive design, use it from your mobile.

Inspiration

Flowork has been inspired by workflowy.

It adapts the concept of being simple to the user yet being complex at the back.

This is another todo list with a slightly simpler philosophy. Hope it helps you stay organised.
